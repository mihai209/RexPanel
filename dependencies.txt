5. Laravel / panel setup commands
---------------------------------

First-time setup:

```bash
cp .env.example .env
php artisan key:generate
touch database/database.sqlite
php artisan migrate --force
composer install
npm install
npm run build
```

One-command setup from composer.json:

```bash
composer run setup
```

If using SQLite:

```bash
mkdir -p database
touch database/database.sqlite
```

8. Extra notes for VPS deployment
---------------------------------

- Web root for Laravel is usually `public/`
- You need writable permissions for:
  - storage/
  - bootstrap/cache/
- If you use SQLite, keep `database/database.sqlite` writable by the PHP user
- If you use websocket features, expose:
  - app HTTP port
  - connector websocket port
  - UI websocket port
- If you use Redis-backed features, install and enable Redis server plus `php-redis`


9. Quick production example
---------------------------

```bash
git clone <your-repo>
cd app
cp .env.example .env
composer install --no-dev --optimize-autoloader
npm install
npm run build
php artisan key:generate
touch database/database.sqlite
php artisan migrate --force
php artisan config:cache
php artisan route:cache
php artisan view:cache
```
