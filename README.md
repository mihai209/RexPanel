# RexPanel Beta [not recommended for production now]

RexPanel built by using Laravel + React and PHP is a faster version of CPanel and better since isn't based on JS/EJS
Yet RexPanel is trying to be better than CPanel Rocky isn't yet stable since is in beta
And if you found any bugs you can make a PR.

RexPanel is compatible with the existing Connector from CPanel Rocky.
